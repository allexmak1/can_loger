# can_loger

Прибор для лога CAN отображения его на экран и записи  трафика.

## Hardware 
 * CPU: STM32F401CC (100 MHz SYSCLK)
 * DISPLAY: DMG85480F050_01WTC
 * CAN driver: TJA1050
 * АКБ: Li-Ion (3,7v)
 * зарядная плата: TP4056
